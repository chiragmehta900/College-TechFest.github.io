Box Cricket

Department : GENERAL

Team Members : Maximum 4 person and minimum 3 person can participate.

Fees per Student : ₹ 50 Per Person

Event Description :

  • BATTING RULES
    • Runs scored by the batsmen will be valid only if he bats with at least one leg inside the batting crease before the ball is delivered.
    • Batting team can score 8 runs by hitting the 8-run board on the field.
  • BOWLING RULES
    • The ball should be pitched beyond the no-ball line, otherwise, it will be called noball.
    • Bowler has to ball within the given box i.e bowling crease.
    • Usual No ball, Wide ball, Byes and Overthrow will be applicable.
    • Sidearm bowling is not allowed.
  • DISMISSALS
    • Stumping on no-ball will not be considered.
    • If the keeper collects the ball before it passes the stumps for the stumping, then stumping will be not valid.
    • Umpire’s decision will be binding and final. No argument will be entertained with the umpire.
  • UNIQUE RULES
    • 3-Dot Ball.(If a batsman fails to score a run for three consecutive deliveries, he/she will be declared out)
    • Runs deducted for a wicket.(5 runs are deducted from the total for each wicket)
    • Jackpot over. (The runs scored in the last over of an inning will be DOUBLED!) • Goal post counted as 8 runs.
